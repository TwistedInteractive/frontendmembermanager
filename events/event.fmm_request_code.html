This event is used to make use of the forgot password feature?

To set it up, make sure you got the Email Template Filter installed, and follow the following steps:

  • Read the tutorial accompanied with the Email Template Filter.
  • Create an email template for password retreival and set it in the preferences page.

An example form could look something like this:

<form method="post" action="">
	E-mail address: <input type="text" name="fields[e-mail-address]" /><br />
	<input type="hidden" name="fields[section]" value="users" />
	<input type="submit" name="action[request-code]" value="Reset password" />
</form>
